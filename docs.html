Documentation

Welcome to the Prompting It documentation. Learn how to build, manage, and deploy production-ready AI prompts at scale.

Introduction

Prompting It is the enterprise prompt operating system that brings version control, approval workflows, and multi-model testing to your AI prompts. Whether you're a solo developer or an enterprise team, our platform helps you ship better prompts faster.

Why Prompting It?

Traditional prompt management leads to scattered files, no version history, and zero visibility into what's working. Prompting It solves this with a purpose-built platform for production AI workflows.

  • Version control for every prompt iteration
  • Built-in A/B testing and performance analytics
  • Team collaboration with approval workflows
  • Multi-model deployment (GPT-4, Claude, Gemini)

Installation

Install the Prompting It SDK in your project:

NPM

npm install @promptingit/sdk

Python

pip install promptingit

Initialize the Client

import { PromptingIt } from '@promptingit/sdk';
                        
                        const client = new PromptingIt({
                          apiKey: process.env.PROMPTINGIT_API_KEY
                        });

API Reference

The Prompting It API allows you to programmatically manage prompts, run tests, and deploy to production.

Authentication

All API requests require an API key. Include it in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Endpoints

Method Endpoint Description
GET /v1/prompts List all prompts
POSTspan> /v1/prompts Create a new prompt
GETspan> /v1/prompts/:id Get prompt by ID
PUTspan> /v1/prompts/:id Update a prompt
DELETEspan> /v1/prompts/:id Delete a prompt
POSTspan> /v1/prompts/:id/execute Execute a prompt

div>
